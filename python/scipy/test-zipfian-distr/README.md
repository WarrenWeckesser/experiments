Various tools and scripts for testing the "Zipfian" distribution
----------------------------------------------------------------

The script `plot_histogram_and_pmf.py` generates plots like this,
allowing visual verification that a large sample agrees with the
expected distribution:

![](https://github.com/WarrenWeckesser/experiments/blob/main/python/scipy/test-zipfian-distr/histogram_and_pmf.png)


The script `test_zipfian_rvs.py` runs many power divergence tests where each
test has a large number of samples.  It can produce a histogram of p-values
such as this:

![](https://github.com/WarrenWeckesser/experiments/blob/main/python/scipy/test-zipfian-distr/power_divergence_histogram.png)


The script `multinomial_test.py` defines functions for running the
[multinomial test](https://en.wikipedia.org/wiki/Multinomial_test) on samples
generated by `zipfian.rvs()`.  For example, here is a run of the
